memcached(client_config: Optional[ClientConfig] = None, cache_threshold: float = 0.3, allow_repeated_training: bool = False, rtol_threshold: float = 0.1, min_repetitions: int = 3, ignore_args: Optional[Iterable[str]] = None)[source]

Decorate a callable with this in order to have transparent caching.

The function’s code, constants and all arguments (except for those in ignore_args are used to generate the key for the remote cache.

FIXME?:

Due to the need to pickle memcached functions, this returns a class instead of a function. This has the drawback of a messy docstring.

Parameters

  • client_config – config for pymemcache.client.Client(). Will be merged on top of the default configuration.

  • cache_threshold – computations taking below this value (in seconds) are not cached

  • allow_repeated_training – If True, models with same data are re-trained and results cached until rtol_threshold precision is reached

  • min_repetitions – minimum number of repetitions

  • ignore_args – Do not take these keyword arguments into account when hashing the wrapped function for usage as key in memcached

Para rtol_threshold

relative tolerance for repeated training. More precisely, memcache will stop retraining the models once std/mean of the scores is smaller than the given rtol_threshold

Returns

A wrapped function

The default configuration is:

default_config = dict(
    server=('localhost', 11211),
    connect_timeout=1.0,
    timeout=0.1,
    # IMPORTANT! Disable small packet consolidation:
    no_delay=True,
    serde=serde.PickleSerde(pickle_version=PICKLE_VERSION)
)
