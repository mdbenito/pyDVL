linear

Contains all functions for closed form solution of influences for standard linear regression.

compute_linear_influences(x: NDArray, y: NDArray, x_test: NDArray, y_test: NDArray, influence_type: InfluenceType = InfluenceType.Up)[source]

Calculate the linear influences of the training set onto the validation set assuming a linear model Ax+b=y. Points with low (or negative) influences are less valuable for model training than higher influence points.

Parameters

  • x – A np.ndarray of shape [MxK] containing the features of input data points.

  • y – A np.ndarray of shape [MxL] containing the targets of input data points.

  • x_test – A np.ndarray of shape [NxK] containing the features of the test set of data points.

  • y_test – A np.ndarray of shape [NxL] containing the targets of the test set of data points.

  • influence_type – Which algorithm to use to calculate influences. Currently supported options: ‘up’ or ‘perturbation’.

Returns

A np.ndarray of shape [BxC] with the influences of the training points on the test points.

Calculate the influences of the training set onto the validation set for a linear model Ax+b=y.

Parameters

  • linear_model – A tuple of np.ndarray’ of shape [NxM] and [N] representing A and b respectively.

  • x_train – A np.ndarray of shape [MxK] containing the features of the train set of data points.

  • y_train – A np.ndarray of shape [MxL] containing the targets of the train set of data points.

  • x_test – A np.ndarray of shape [NxK] containing the features of the test set of data points.

  • y_test – A np.ndarray of shape [NxL] containing the targets of the test set of data points.

Returns

A np.ndarray of shape [BxC] with the influences of the training points on the test points.

Calculate the influences of each feature of the training set onto the validation set for a linear model Ax+b=y.

Parameters

  • linear_model – A tuple of np.ndarray’ of shape [NxM] and [N] representing A and b respectively.

  • x_train – A np.ndarray of shape [MxK] containing the features of input data points.

  • y_train – A np.ndarray of shape [MxL] containing the targets of input data points.

  • x_test – A np.ndarray of shape [NxK] containing the features of the test set of data points.

  • y_test – A np.ndarray of shape [NxL] containing the targets of the test set of data points.

Returns
