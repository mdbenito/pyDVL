init_parallel_backend(config: ParallelConfig) → RayParallelBackend[source]

Initializes the parallel backend and returns an instance of it.

Parameters

config – instance of ParallelConfig with cluster address, number of cpus, etc.

Example

>>> from pydvl.utils.parallel.backend import init_parallel_backend
>>> from pydvl.utils.config import ParallelConfig
>>> config = ParallelConfig(backend="ray")
>>> parallel_backend = init_parallel_backend(config)
>>> parallel_backend
<RayParallelBackend: {'address': None, 'num_cpus': None}>
