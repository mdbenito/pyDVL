class RayActorWrapper(actor_handle: ObjectRef, parallel_backend: RayParallelBackend)[source]

Bases: object

Wrapper to call methods of remote Ray actors as if they were local.

Taken almost verbatim from: https://github.com/JaneliaSciComp/ray-janelia/blob/main/remote_as_local_wrapper.py

Example

>>> from pydvl.utils.parallel import init_parallel_backend
>>> from pydvl.utils.config import ParallelConfig
>>> from pydvl.utils.parallel.actor import RayActorWrapper
>>> class Actor:
...     def __init__(self, x):
...         self.x = x
...
...     def get(self):
...         return self.x
...
>>> config = ParallelConfig()
>>> parallel_backend = init_parallel_backend(config)
>>> actor_handle = parallel_backend.wrap(Actor).remote(5)
>>> parallel_backend.get(actor_handle.get.remote())
5
>>> wrapped_actor = RayActorWrapper(actor_handle, parallel_backend)
>>> wrapped_actor.get()
5
